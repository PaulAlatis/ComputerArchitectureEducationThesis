# ComputerArchitectureEducationThesis
A thesis project
It contains 3 different games
Each game trains people on different aspects of computer architecture and logic circuits classes

1st game
Binary Stars 
A space invaders type game with soft bullet hell traits, It also adds binary to decimal conversion learning.


2nd game
Quiz (name change pending)
A card battler game with with quiz elements during battle.The questions are based on exams from the University of West Attica class of "Computer Architecture"
